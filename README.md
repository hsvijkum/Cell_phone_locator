# Cell phone detector
Goal: A simple program tasked with finding the location of a phone dropped on the floor from a single RGB camera image given a small training dataset of 100 images. You can use any method for detecting phone positions that you feel is suited to the problem. In order to meet customer's requirements, the program has to correctly detect the location of the phone in 70% of the images.
